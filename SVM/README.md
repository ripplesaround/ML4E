### 综述

- 二分类模型
- 基本模型定义：**在特征空间上的<u>间隔最大</u>的<u>线形分类器</u>**
  - 间隔最大-》有别于感知机
  - 核技巧-》实质上的非线形分类器
- 学习策略：间隔最大化，可形式化为求解一个凸二次规划的问题，夜灯驾驭正则化的**合页损失函数**

- 学习算法：求解凸二次规划的最优化算法。

#### SVM有很多类：

- 线性可分SVM / 硬间隔SVM
  - 线形分类器，硬间隔最大化
- 线性SVM / 软间隔SVM
  - 软间隔最大化（数据**近似**线形可分），线形分类器
- 非线性SVM
  - 数据线性不可分，核技巧+软间隔最大化
